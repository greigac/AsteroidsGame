About

Asteroids is a canvas game created using HTML5 and Javascript. Objective of the game is to destroy asteroids and stay alive. You have 3 lives. Use them wisely as its game over when you run out! Avoid the asteroids. Especially the one with the mustache! It shoots lasers!

Controls

To play use the thrust to move forward. Left and right to rotate the ship and fire to shoot the laser.
